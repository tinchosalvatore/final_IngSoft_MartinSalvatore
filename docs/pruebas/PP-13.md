Procedimiento de Prueba - Martin Salvatore

CP 11.1.1 — Listar usuarios exitosamente

1. Iniciar sesión con el usuario "jperez".
2. Verificar en la base de datos que existen usuarios que comparten más de 2 amigos con "jperez".
3. Dirigirse a la sección de buscador de usuarios con amigos en común.
4. Confirmar que el sistema realiza la petición GET /usuarios?amigosEnComun=2.
5. Verificar que el sistema muestra la lista de usuarios recuperada.
6. Comprobar que todos los usuarios de la lista tienen efectivamente más de 2 amigos en común con "jperez".

CP 11.1.2 — Sin resultados

1. Iniciar sesión con un usuario que no tenga otros usuarios con más de 2 amigos en común.
2. Dirigirse a la sección de buscador de usuarios con amigos en común.
3. Confirmar que el sistema realiza la petición GET /usuarios?amigosEnComun=2.
4. Verificar que el sistema lanza UsuarioNotFoundException y retorna 404.
5. Comprobar que se muestra el mensaje de error "No se encontraron usuarios".
